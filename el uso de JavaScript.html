¿Qué es JavaScript?

JavaScript es un lenguaje de programación interpretado que permite implementar funciones complejas en páginas web, tales como mostrar contenido dinámico, animaciones y procesamiento de datos en tiempo real. Es uno de los tres pilares del desarrollo web, junto con HTML y CSS.

Cómo utilizar JavaScript en una página web

Para incluir JavaScript en una página web, puedes usar la etiqueta <script>. Este código puede colocarse en la cabecera (<head>) o antes del cierre del cuerpo (</body>) del documento HTML.

<script>
alert('Hola, mundo!');
</script>

Manejo de Variables en JavaScript

JavaScript permite declarar variables utilizando las palabras clave var, let o const. Las variables son espacios de almacenamiento que pueden contener diferentes tipos de datos, como números, cadenas de texto, objetos, y más.

Ejemplo:

let nombre = 'Juan';
let edad = 25;
const PI = 3.1416;
