¿Qué es JavaScript?

JavaScript es un lenguaje de programación que permite añadir interactividad a las páginas web. Se ejecuta en el navegador del usuario y permite crear dinámicas avanzadas como animaciones, validaciones de formularios, carga dinámica de contenido, entre otras cosas.

Cómo utilizar JavaScript en una página web

JavaScript puede incluirse en una página web de varias maneras. La más común es a través de una etiqueta <script> dentro del documento HTML. Aquí un ejemplo básico:

<script>
    console.log("Hola, JavaScript!");
</script>

Este código imprimirá "Hola, JavaScript!" en la consola del navegador.

Manejo de Variables en JavaScript

En JavaScript, las variables se pueden declarar utilizando var, let, o const. Dependiendo del contexto, cada una tiene un uso específico.

let nombre = "Juan";
const edad = 25;
var ciudad = "Madrid";

En el ejemplo anterior, declaramos una variable nombre con valor "Juan", una constante edad con valor 25, y una variable global ciudad con valor "Madrid".
